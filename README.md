# Relay Helper

This repository hosts the official Relay Helper binaries.

## Download

Downloads are available on the [Releases page](https://github.com/dotrly/dist/releases).

## Installation

### macOS / Linux
```bash
# curl -fsSL https://relay.dotrly.com/install.sh | bash
curl -fsSL https://relay.notapublicfigureanymore.com/install.sh | bash
```

### Windows
```powershell
# irm https://relay.dotrly.com/install.ps1 | iex
irm https://relay.notapublicfigureanymore.com/install.ps1 | iex
```
